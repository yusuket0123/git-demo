### Description of Branch ###
- Master (Main) branch: master copy
- HEAD: current location that it is being checked out
- check a branch:
    git branch (* is current location)
- make a new branch:
    git brach <branch-name>
    * make a neww branch based upon the current HEAD
- switch between branches:
    git switch <branch-name>
    git checkout <branch-name>

Tips
- create a new branch and switch to it at once
    git switch -c <branch-name>
- delet branch: git branch -d
