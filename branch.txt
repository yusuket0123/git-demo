### Description of Branch ###
0. Check a branch:
    - git branch (* is current location)
    * -d: delete branch
    * -c: copy branch
    * -f: reset branch to startpoint
    * -m: rename branch name (after change to the branch)
1. Make a new branch:
    git brach <branch-name>
    * make a neww branch based upon the current HEAD
2. Switch between branches:
    - git switch <branch-name>
    - git checkout <branch-name>
    *  -c: create a new branch and switch to it at once

Tips
- Master (Main) branch: master copy
- HEAD: current location that it is being checked out
